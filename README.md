## Overview

An interactive loan portfolio analytics dashboard built with Streamlit and Plotly,
demonstrating end-to-end data pipeline and analytics infrastructure for a Nigerian
digital lending company.

The dashboard provides real-time visibility into loan portfolio performance and
recovery operations, the kind of analytics infrastructure that enables data-driven
decisions in fintech lending operations.

## Features

### 📋 Portfolio Report
- 18 KPI cards covering disbursement, repayment, arrears and balances
- Loan count gauge with portfolio health indicator
- Previous vs Current loan classification comparison (pie charts)
- Top 10 states by average loan disbursed (bar chart)
- Disbursement breakdown by lending channel
- Lost % by officer, accountability table
- PAR Principal by classification
- Full portfolio detail table (40,000 records)

### ⚠️ Recovery Dashboard
- 18 KPI cards covering overdue, recovery and inflow metrics
- Overdue and recovery trend over time (line chart)
- Cumulative overdue growth chart
- Expected vs Overdue vs Recovered by channel (grouped bar)
- Overdue under recovery by classification (donut chart)
- Recovery breakdown by State, Region, Channel, and Cluster
- Full overdue accounts table with recovery rates

### 🎛️ Interactive Filters
All charts and KPIs respond dynamically to sidebar filters:
- Channel (Salary Credit, Asset Finance, Retail Credit, Mobile Advance)
- State (all 37 Nigerian states and FCT)
- Loan Status (Active, Completed, Defaulted)
- Classification (Performing → Lost)
- Region (6 geopolitical zones)

## Dataset

The dashboard runs on a synthetic dataset of 40,000 loan records generated
programmatically to mirror Nigerian fintech lending data characteristics.

| Property | Value |
|---|---|
| Total records | 40,000 loans |
| Channels | Salary Credit, Asset Finance, Retail Credit, Mobile Advance |
| States | All 37 Nigerian states + FCT |
| Date range | January 2023 – May 2026 |
| Generated by | `generate_data.py` (reproducible, seed=42) |

All customer names, loan IDs, and financial figures are entirely synthetic.

## Run Locally

```bash
git clone https://github.com/gpfalade/loan-portfolio-analytics
cd loan-portfolio-analytics
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Cloud

1. Fork this repository
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Click **New app** → select this repository
4. Main file: `app.py`
5. Click **Deploy**
